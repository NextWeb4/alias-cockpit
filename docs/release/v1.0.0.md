# Alias Cockpit v1.0.0

Windows x64 local email alias cockpit.

## Release Assets

- `AliasCockpit-win-x64-setup.exe` - WiX Burn setup EXE that embeds the MSI.
- `AliasCockpit-win-x64.msi` - per-machine MSI installer.
- `AliasCockpit-win-x64-portable.zip` - portable folder publish archive.

## Verification

- Built and packaged with `scripts\verify-release.ps1`.
- Gate coverage: build, tests, benchmark, format verification, publish, publish pruning, launch smoke, portable zip check, MSI validation, setup EXE extraction check, and UI smoke.
